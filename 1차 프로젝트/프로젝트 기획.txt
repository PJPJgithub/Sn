어디에 도움이 되는가 : 운전자가 운전을 하면서 실시간으로 주식을 매매할 수 있다. 지체없이 매수/매도 결정에서부터 실행까지 할 수 있다.
기존 앱과의 차이점(상품성, 비슷한 기능의 앱) : 음성으로 실시간으로 주식을 매매할 수 있다.
페르소나

UI가 변하는 기능 추가 - automation 하기 편함
UI로 해당 기사의 이미지와 헤드라인을 UI로 띄워줌. (urltoimage, title)

[기능 정리]
1.음성인식 명령어
-'키워드'뉴스 알려 줘 : 해당 뉴스의 가장 최신 뉴스를 불러와 띄워 준다.
-다음 : 처음 말한 키워드의 다음 뉴스를 불러와 띄워 준다.
-이전 : 처음 말한 키워드의 이전 뉴스를 불러와 띄워 준다.
-내용 : 해당 기사의 상세 내용을 말해준다.
-홈으로 : 검색했던 기사 목록 및 모든 정보를 초기화한다.

2.버튼
-홈 : 초기화
-음성인식 : 음성인식 시작
-목록 : 검색했던 기사 목록 보여줌(헤드라인, 검색한 시간)
-이전 : 해당 키워드의 이전 기사 보여줌.
-다음 : 해당 키워드의 다음 기사 보여줌.
-화면의 기사 : 해당 기사의 상세 내용 보여줌.
-목록의 헤드라인 칸 : 해당 기사의 상세 내용 보여줌.
-뒤로가기 : 뒤로감

3.기능
-api로 최신 뉴스 받아옴
-뉴스 저장 기능
