<main activity>
마이크 버튼 클릭 -> 음성 인식 keyword 입력 -> api 받아와서 띄워주면 -> 기사이미지, 헤드라인 잘 뜨는지 체크
기사 불러온 후에 -> 버튼 동작 잘 하는지 테스트

<List activity>
다음 버튼으로 기사 추가(음성인식 or 버튼 클릭) ->리스트 버튼 클릭 -> 기사들 잘 추가 됐는지 확인(시간 순서 정렬)

<Detail activity>
main activity에서 기사를 눌렀을 때 -> detail page잘 뜨는 지 확인
리스트에서 헤드라인 눌렀을 때 -> detail page 잘 뜨는 지 확인

reset 버튼 눌렀을 때 -> 기사 비우고, 리스트 비우는지 테스트
