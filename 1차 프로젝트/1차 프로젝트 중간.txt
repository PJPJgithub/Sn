[프로젝트 주제]
이름 : SnapNews(가칭)
Target : 운전자
왜 필요한가: 
	- 운전하면서 자신이 원하는 키워드와 관련된 뉴스를 들을 수 있다.
어디에 도움이 되는가: 
	- 관심있는 뉴스만 들을 수 있고 주/정차 시 검색했던 목록 중 보고 싶었던 기사를 편하게 자세히 볼 수 있다.
기존 앱과의 차이점:
	- 비슷한 기능의 앱 : 라디오, DMB 등 뉴스를 알려주는 앱
	- 상품성/차이점 : 원하는 키워드만 모아서 빠르게 핵심만 들을 수 있는 차량용 뉴스 앱

[페르소나]
{
	운전자가 운전을 하고 있다. 
	"GM 뉴스 알려 줘". GM과 관련된 최신 뉴스를 알려준다. 
	계속 듣고 싶다. "계속 알려 줘". "..." 
	다른 뉴스가 듣고 싶다. "테슬라 뉴스 알려 줘". 
	테슬라의 최신 뉴스를 알려준다...
}

[기능 정리]
1.음성인식 명령어
-'키워드'뉴스 알려 줘 : 해당 뉴스의 가장 최신 뉴스를 불러와 기사 이미지와 헤드라인을 띄워 준다.
-다음 : 인식된 키워드의 다음 뉴스를 불러와 띄워 준다.
-이전 : 인식된 키워드의 이전 뉴스를 불러와 띄워 준다.
-내용 : 해당 기사의 상세 내용을 말해준다.
-계속 : 인식된 키워드의 다음 뉴스를 10초의 간격으로 자동으로 계속 알려준다.
-초기화 : 검색했던 기사 목록 및 모든 정보를 초기화한다.
-종료 : 앱 종료

2.버튼
-리셋 : 초기화
-음성인식 : 음성인식 시작
-목록 : 검색했던 기사 목록 보여줌(헤드라인, 검색한 시간)
-목록의 헤드라인 칸 : 해당 기사의 상세 내용 보여줌.
-이전 : 인식된 키워드의 이전 기사 보여줌.
-다음 : 인식된 키워드의 다음 기사 보여줌.
-화면의 기사 : 해당 기사의 상세 내용 보여줌.
-뒤로가기 : 뒤로감

3.주요 기술
-newsapi.org의 api 이용 최신 뉴스 데이터 처리
-stt
-tts

4.확장성
- 뉴스 저장 기능(하트)
- 뉴스 공유 기능
- 뉴스 요약/정리/분석(AI)
- ...
